Projects

Clinical Data Analysis using NER

Built NER using BERT and Bi-LSTM model to identify Disease and Drug entities

This project focuses on extracting critical information from handwritten doctor prescriptions using image processing, Optical Character Recognition (OCR), and Natural Language Processing (NLP). The challenge lies in real-time deployment and performance, addressing issues such as manual query generation and OCR limitations. The goal is to automate the extraction and categorization of information like drugs and diseases. The extracted text is processed with Named Entity Recognition (NER) and contextualized using Bidirectional LSTM and BERT models to enhance understanding of medical entities. By integrating Machine Reading Comprehension (MRC), the system improves query generation and overall comprehension, aiming to optimize clinical data interpretation.

Customer Segmentation

Applied ML classification algorithms and NLP for effective customer grouping

Our project classifies customers into distinct clusters to enable targeted marketing strategies for maximizing profit. As consumer data grows, efficient clustering algorithms are crucial for processing large datasets. K-Means offers faster computation and better performance for large datasets, though it requires predefined "K" clusters. In contrast, hierarchical clustering doesn’t need predefined clusters and handles smaller datasets well, providing more accurate results with dendrograms. While K-Means is ideal for large-scale data, hierarchical clustering excels with smaller datasets and provides better accuracy as the number of clusters increases.
